#Project Overview
This repository contains the front-end development of the Josh Property website, created for a realty marketing brand. The project focuses on delivering a clean, professional, and conversion-driven real estate platform. The goal was to build a modern property website that highlights listings clearly, communicates trust, and provides a smooth user experience across devices.
The development prioritizes performance, responsiveness, and straightforward navigation to ensure potential buyers, sellers, and investors can easily explore property information.

## Key Objectives
- Build a modern real estate marketing website
- Present property listings in a clear and organized layout
- Ensure responsive design for desktop, tablet, and mobile
- Maintain clean, scalable front-end structure
- Focus on usability and fast loading performance

## Technical Stack
- Front-end Framework: React
- Languages: HTML, CSS, JavaScript (.js files)
- Styling: Tailwind CSS
- Tooling: Visual Studio Code
- Version Control: Git & GitHub

## Design & Development Principles
- User-focused layout: Clear property sections and structured information hierarchy
- Professional aesthetic: Clean design that builds trust and credibility
- Responsive design: Fully optimized across all modern devices
- Component-based structure: Reusable React components for scalability
- Maintainable codebase: Organized file structure and readable naming conventions

## Project Scope
- Front-end real estate marketing website
- Static property listings (no backend integration)
- Optimized for modern browsers
- Suitable for realtors, agencies, and property marketers

Active development / portfolio project
